# Halftone Studio

Web-based image and video effects editor with Material Design 3 interface.

## Features

- **6 Visual Styles**: Classic, RGB Dots, LED Glow, Blocks, ASCII Art, Duotone
- **Image Processing**: Upload PNG, JPG, GIF images up to 30MB
- **Video Processing**: Upload MP4, WebM videos up to 30MB
- **Export Options**: PNG, GIF animation, WebM video
- **Real-time Preview**: All changes visible instantly
- **Bilingual**: English and Russian interfaces
- **Responsive Design**: Works on desktop and mobile devices

## Usage

1. Open `index.html` in a browser
2. Upload an image or video
3. Select a visual style
4. Adjust parameters (density, size, contrast)
5. Enable animation for pulsating effects
6. Export result as PNG, GIF, or video

## Files

- `index.html` - Main page structure
- `style.css` - Styles and responsive design
- `script.js` - Application logic
- `video.js` - Video processing module

## Technologies

- Tailwind CSS (CDN)
- gif.js (CDN)
- Vanilla JavaScript
- Canvas API

## Deployment

Deploy to Vercel:
1. Push to GitHub repository
2. Connect repository to Vercel
3. Vercel auto-detects static site
